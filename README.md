# Conways-Game-Of-Life-Fast

## Building 
```
mkdir build
cd build
cmake ..
make
```

## Running
```
cd bin
python3 Gen.py
./cgol [total number] [generations] [1/0 save]
```

## About
This is a re-do of a previous project. Moving away from OOP and 
fixing memory leaks as well as a focus on improved performance.
